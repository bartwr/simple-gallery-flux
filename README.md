# Simple gallery flux

Outdoor Medicine version.

## Required on live setup

First install and build locally.

```sh
npm install && bower install && npm run build
```

Then copy (at least) these files to the server.

* `bower_components/*`
* `dist/*`
* `plugin.php`

## Develop

Always:

    npm run start

First time:

    npm install && bower install && npm run start
